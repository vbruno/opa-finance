
# Visão Técnica do Sistema

## Frontend
- Vite + React + TS  
- TanStack Router  
- React Query  
- Tailwind + shadcn/ui  
- Axios + Zod  

## Backend
- Fastify + TypeScript  
- Drizzle ORM  
- PostgreSQL  
- JWT Auth  
- Zod  
- Dashboard com top gastos por categoria/subcategoria  

## Infraestrutura
- Docker + Portainer  
- Nginx  
- Certbot  
- VPS Hostinger  

## Arquitetura Backend
- Routes  
- Controllers  
- Services  
- Repositories  
- Schemas Zod  

## Rotas de Sistema
- `/health`: health check da API  
- `/`: status simples da API  

## Autenticação
- Access Token curto  
- Refresh Token longo (httpOnly + secure)  
- Backend stateless
