
# Visão Macro do Projeto

## Objetivo
Permitir que o usuário visualize onde está gastando e identificar oportunidades de reduzir despesas.

## Público-Alvo
- Uso pessoal
- Pessoas que desejam reduzir gastos

## Problemas Resolvidos
1. Falta de visibilidade dos gastos  
2. Dificuldade de controlar o mês  
3. Gastar mais do que ganha  
4. Falta de clareza por categoria  
5. Desorganização financeira  
6. Histórico difícil de analisar  
7. Sem comparação entre meses  

## MVP
- Autenticação
- Contas
- Categorias
- Subcategorias
- Transações
- Transferências entre contas
- Dashboard
