
# Módulo: Transactions

## Responsabilidades
- Criar transações
- Editar transações
- Excluir transações
- Listar transações por filtros

## Regras
- category_id obrigatório
- subcategory_id opcional
- mudar categoria → limpa subcategoria
- valores entram no saldo da conta

## Campos
- amount, type, date, description
